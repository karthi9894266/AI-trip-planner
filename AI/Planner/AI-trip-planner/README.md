# 🌍 AI Trip Planner

An intelligent travel planning application that helps users create personalized trip itineraries using AI. The platform allows users to explore destinations, manage trips, search for flights and hotels, and receive travel recommendations through an interactive chatbot.

## 🚀 Features

* 🔐 User Authentication (Login & Registration)
* 🤖 AI-Powered Trip Itinerary Generation
* ✈️ Flight Search
* 🏨 Hotel Search
* 💬 Interactive Travel Chatbot
* 📅 Trip Planning & Management
* 📍 Destination Recommendations
* 📊 Trip Dashboard

## 🛠️ Tech Stack

### Frontend

* React.js

* Node.js

* MongoDB

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/karthi9894266/AI-trip-planner.git
```

### Frontend Setup

```bash
cd client
npm install
npm run dev
```

### Backend Setup

```bash
cd server
npm install
npm start
```

## 🔑 Environment Variables

Create `.env` files in both `client` and `server`.

Example:

```env
MONGODB_URI=your_mongodb_uri
OPENAI_API_KEY=your_api_key
AMADEUS_API_KEY=your_api_key
JWT_SECRET=your_secret
```
